# Movie App

In order to run the application follow these instructions:

- Pull the repo locally
- Run these commands on your terminal:
- `npm install -g pnpm`
- `pnpm i`
- And finally to run it, type in this command:
- `pnpm dev`
- On your browser go to: http://localhost:5173/
